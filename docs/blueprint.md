# **App Name**: GitHub Pages Portfolio

## Core Features:

- Макет Портфоліо: Чистий та сучасний макет портфоліо, оптимізований для відображення на GitHub Pages.
- Відображення Проєктів GitHub: Інтеграція з GitHub API для автоматичного отримання та відображення останніх проєктів.
- Інтеграція з GitHub Pages: Легке розгортання на GitHub Pages з автоматизованим процесом збірки.
- AI Project Summarizer: Інструмент на основі штучного інтелекту, який генерує професійні резюме проєктів GitHub, включаючи ключові технології, використані та цілі проєкту, для відображення у вашому портфоліо.

## Style Guidelines:

- Основний колір: темно-синій (#34495E), щоб передати професіоналізм та стабільність.
- Колір фону: світло-сірий (#ECF0F1) для чистого, ненав'язливого фону.
- Колір акценту: бірюзовий (#2ABC9B), щоб виділити інтерактивні елементи та заклик до дії.
- Шрифт заголовків: 'Space Grotesk' (sans-serif) для сучасного, технологічного відчуття; шрифт тіла: 'Inter' (sans-serif) для читабельності.
- Використовуйте мінімалістичні, лінійні піктограми для категорій проєктів та навичок.
- Адаптивний дизайн з акцентом на читабельність та візуальну ієрархію. Використовуйте чіткі розділи для вступу, проєктів та контактної інформації.
- Тонкі анімації наведення на картки проєктів та кнопки для забезпечення візуального зворотного зв'язку.
